# SocialeX

SocialeX is a full-featured social media platform where users can:

- 📸 Share posts, photos, videos, and stories  
- ❤️ Like, comment, and engage with other users  
- 🔔 Receive real-time notifications  
- 💬 Chat via real-time messaging (Socket.io)  
- 🧑‍🤝‍🧑 Discover and follow new users  

Built with the MERN stack and modern development tools.

## Tech Stack
- **MongoDB** – Database
- **Express.js** – Backend framework
- **React.js** – Frontend library
- **Node.js** – Server runtime
- **Socket.io** – Real-time communication
